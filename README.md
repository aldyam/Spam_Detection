# Deskripsi

Proyek ini adalah *text classification* untuk mendeteksi apakah sebuah SMS termasuk "Spam" atau "Ham" (bukan spam). Model dilatih menggunakan dataset SMS Spam Collection dari Kaggle.

# Teknologi

* Python
* Pandas & NumPy
* Scikit-learn (TF-IDF, Naive Bayes)
* TensorFlow/Keras (Deep Learning)
* NLTK
